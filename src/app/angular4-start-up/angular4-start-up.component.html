Angular 4 is one of the most popular and powerful javascript frameworks for creating web, mobile, native mobile, and native desktop applications. Angular 4 provides the tools necessary for code reuse and ensures adaptability.

1. Visit Node.js and download.


2. Go to your environment variables and attach Node path into PATH.


3. Visit Visual Studio Code and download.


4. Open VSCode, open cmd.exe, go to proper directory for projects and run the command: "npm install -g @angular/cli"

1. Within the terminal, generate a new project by running the command: "ng new my-app"


2. Once it is finished, enter the directory and launch the server with the commands: "cd my-app" and "ng serve"


3. Visit http://localhost:4200/
